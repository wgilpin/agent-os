# Quickstart: Consent Screen UI

## How to Run Locally

1. Start the Phoenix application locally:
   ```bash
   mix phx.server
   ```
2. Open your browser and navigate to:
   ```text
   http://localhost:4000/consent?manifest=manifests/discovery.md
   ```

## How to Run Tests

Run the newly created LiveView integration tests:
```bash
mix test test/agent_os_web/consent_live_test.exs
```
